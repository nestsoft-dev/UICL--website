# Bayelsa Bio-Data Form (React + Vite)

Production-ready React + TypeScript form that matches the "Bayelsa Bio-Data Form" PDF layout.

## Setup

```bash
npm install
npm run dev
```

## Notes

- Data persists to local storage while typing.
- Submissions are stored under `bayelsa_biodata_submissions`.
- JSON export is available on the success page.
